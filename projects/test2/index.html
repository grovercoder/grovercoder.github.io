Magento E-Commerce / ERP Integration

Description Link to heading

The company had a number of physical stores and warehouses. An E-Commerce solution was needed to integrate with the inventory and order fulfillment system the physical locations used.

The organization had 50,000 products, each with their own set of properties that might change periodically. They also had a very active sales team who would do weekly specials, and various promotions.

Ensuring the website would see fast load times, yet have accurate information and provide a seamless user experience was a bit of a challenge to say the least.

Through the use of an intermediate database, a property mangement tool, the usual web caching tricks, some custom coding for working with the ERP, we were able to see sub 3 second load times, sometimes as low as 1 second. (By today’s standards, this should be even faster.)

This was a great project with a great team scattered across the country.
